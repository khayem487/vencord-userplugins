# PdfViewer

Inline PDF viewer for Discord messages.

## Features

- Preview / Hide PDF
- Scroll all pages
- Fit-width + zoom (`+` / `-`)
- `Ctrl + wheel` / touchpad pinch zoom

## Install

Copy this folder to:
`src/userplugins/pdfViewer`

Then run:

```powershell
corepack pnpm build
corepack pnpm inject
```
